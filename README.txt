###########
DESCRIPTION
###########
Formula 15-112 is a 2D top-down view car racing game based upon the auto racing
sport Formula 1. The player controls a car and races AI opponents around the 
track. 

##########
HOW TO RUN
##########
run main.py

################
LIBARIES/MODULES
################
run pip install -r requirements.txt

########
COMMANDS
########
w - accelerate the player car
a - turn left
d - turn right

# SHORTCUT
p - toggles pausing the game
